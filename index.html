🎓 Entrenar Modelo Personalizado

Entrena un modelo CNN con 7,500+ imágenes de los 151 Pokémon para obtener 75-85% de precisión en el reconocimiento.

1 Descargar dataset (~500MB)
2 Entrenar modelo (30-60 min)
3 Listo para usar

🖥️ Entrenamiento Automático:

training/train-auto.bat Windows
training/train-auto.sh Linux/Mac

📝 Comandos Manuales:

cd training
npm install
node download-images.js
node create-model.js
node train.js

⚠️ Requisitos:

  • Node.js 16+
  • 8GB RAM mínimo (16GB recomendado)
  • ~2GB espacio en disco
  • GPU opcional pero recomendada
